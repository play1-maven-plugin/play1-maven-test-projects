Authorised

Each of the following links should lead to a valid page

  • Unrestricted method in a restricted controller
  • A method in a restricted controller that is both restricted and unrestricted
  • An unannotated method in an @Unrestricted controller that extends a restricted controller. Normally, this method would inherit its restrictions from the parent controller.


Unauthorised

Each of the following links should lead to a forbidden page

  • Unannotated method in a restricted controller
  • Restrict-annotated method in a restricted controller
  • A restricted method in an @Unrestricted controller.
